# Twitter Algorithm Optimizer

**Twitter Algorithm Optimizer** 是一个基于 Twitter 开源算法洞察的技能，旨在分析并优化推文内容，以获得最大的传播范围和用户参与度。

## 核心理念

- **算法透明化**：利用 Twitter 开源的算法机制（如 Real-graph, SimClusters），揭秘内容分发的底层逻辑。
- **参与度导向**：通过优化互动信号（点赞、回复、转发、书签），提升推文在信息流中的排名。
- **社区共鸣**：强调内容与特定社区（SimClusters）的匹配，而非盲目追求泛化流量。

## 核心功能与工作流

1. **推文草稿分析**：根据 Twitter 的排名机制评估推文表现，识别潜在的算法风险。
2. **内容优化重写**：针对性地调整推文措辞、结构和互动触发点（CTA），提升排名权重。
3. **分发策略建议**：提供基于 Real-graph（追随者互动）和 TwHIN（话题匹配）的发布策略。
4. **负面信号规避**：识别并避免可能导致降权、静音或举报的敏感内容。

## 技能库概览

- **核心算法模型**：涵盖 Real-graph (互动预测)、SimClusters (社区探测)、TwHIN (主题图谱) 和 Tweepcred (账号权重)。
- **互动信号图谱**：详细解析显性信号（点赞、回复、转发）与隐性信号（点击、停留时长）的权重。
- **行业优化模版**：提供针对开发者、产品发布、观点分析等场景的优化方案。
