# Tavily Search

**Tavily Search** 是一个利用 Tavily API 为 AI Agent 优化的网络搜索技能。它旨在提供干净、相关且非爬虫感知的搜索结果，非常适合 AI 智能体直接消费。

## 标签

文档与办公 | 已验证

## 核心理念

- **AI 优化**：不同于传统的搜索引擎，Tavily 专为 AI 训练和智能体设计，返回的结果更符合大模型的处理需求。
- **深度搜索能力**：支持深度研究（--deep）模式，能够跨多个来源获取更全面的信息。
- **内容提取**：除了搜索，还内置了网页内容提取功能，方便 Agent 直接阅读目标页面。

## 核心功能与工作流

1. **智能搜索**：支持通用搜索和新闻专项搜索（--topic news），可自定义返回结果数量。
2. **深度研究**：通过 `--deep` 参数启动更高级的搜索模式，适合复杂问题的调研。
3. **网页内容提取**：通过 URL 直接提取网页的纯净文本内容。
4. **轻量化输出**：返回精简的相关片段，减少 Agent 的上下文负担。

## 技能库概览

- **搜索脚本**：`search.mjs` - 核心搜索逻辑，处理查询和参数。
- **提取脚本**：`extract.mjs` - 网页文本提取工具。
- **配置管理**：依赖 `TAVILY_API_KEY` 环境变量。
