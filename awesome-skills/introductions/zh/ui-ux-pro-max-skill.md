# UI UX Pro Max

**UI UX Pro Max** 是一个为 AI 提供设计智能体的技能，用于在多个平台和框架上构建专业的 UI/UX 界面。

## 标签

设计与创意 | 已验证

## 核心理念

- **设计系统生成**：基于 AI 推理引擎，分析项目需求并自动生成完整的定制化设计系统
- **行业特定规则**：内置 100 条行业特定推理规则，覆盖 SaaS、金融、医疗、电商等领域
- **标准化设计语言**：提供 67 种 UI 风格、96 种配色方案、57 种字体组合的专业设计资源

## 核心功能与工作流

1. **设计系统生成器**：输入产品描述（如"为美容水疗中心构建落地页"），AI 自动推荐完整的模式、风格、配色、字体和效果
2. **智能匹配引擎**：并行搜索产品类别、风格推荐、配色方案、落地页模式和字体配对，提供最佳设计组合
3. **交付前检查清单**：自动验证对比度、响应式、无障碍访问等 99 条 UX 指南和反模式规则
4. **多技术栈支持**：支持 React、Next.js、Vue、Nuxt.js、Svelte、Astro、SwiftUI、Jetpack Compose、React Native、Flutter 等 13 种技术栈

## 技能库概览

- **67 种 UI 风格**：包括极简主义、新拟物、玻璃拟物、粗野主义、Bento Grid、暗黑模式、AI 原生 UI 等
- **96 种配色方案**：针对 SaaS、电商、医疗、金融科技、美容等行业的专用配色
- **57 种字体配对**：精选的 Google Fonts 字体组合，带导入链接
- **100 条推理规则**：覆盖科技、金融、医疗、电商、服务、创意、新兴技术等行业的设计规则
- **25 种图表类型**：适用于仪表板和数据分析的图表推荐
