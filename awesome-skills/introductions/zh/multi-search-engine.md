# Multi Search Engine

**Multi Search Engine** 是一个无需 API Key 的搜索引擎集成工具，聚合了 8 个国内和 9 个国际搜索引擎，并支持高级搜索语法与隐私保护。

## 核心理念
- **无 API 密钥依赖**：直接通过 Web 请求进行数据抓取提取，降低使用门槛。
- **全方位覆盖**：集成涵盖国内及全球的主流引擎，满足多样化及特定地区的信息搜索需求。
- **隐私优先**：原生支持 DuckDuckGo、Startpage 等隐私保护搜索引擎，保障追踪安全。
- **高阶检索能力**：支持高级搜索操作符与时间过滤器，帮助进行更精准的信息定位与筛选。

## 关键特性与工作流
1. **多引擎聚合检索**：提供多达 17 种国内外主流搜索引擎的即席访问。
2. **高级搜索操作符**：支持使用 `site:`（站内查询）、`filetype:`（指定文件格式）、`-`（排除关键词）、`""`（精确匹配）及 `OR` 进行复杂查询。
3. **时间过滤器**：允许按照指定时间范围（如过去一小时 `tbs=qdr:h`，一天，一周，一月，一年）精确筛选搜索结果。
4. **DuckDuckGo Bangs 快捷键**：支持使用 `!g`、`!gh` 等简短前缀快速跳转至特定平台的生态搜索。
5. **WolframAlpha 知识计算**：直接支持数学积分计算、各种单位（如汇率）换算、实时天气等知识性问题的精准计算与解答。

## 技能库概览
- **国内搜索引擎**：涵盖百度、必应中国、360、搜狗、微信、头条、集思录。
- **国际搜索引擎**：涵盖 Google、Google HK、Bing 国际版、Yahoo。
- **隐私保护引擎**：涵盖 DuckDuckGo（无追踪限制）、Startpage（Google 结果 + 隐私）、Brave（独立索引）、Ecosia 和 Qwant（符合欧盟 GDPR）。
- **知识与智能计算**：WolframAlpha（涵盖数学、转换、股票与天气）。
