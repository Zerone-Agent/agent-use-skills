# Prompt Engineering

**Prompt Engineering** is a set of advanced prompt patterns and best practices specifically designed for AI Agents. It integrates Anthropic's official recommendations, persuasion psychology principles, and production-grade LLM interaction patterns.

## Core Philosophy

- **Control & Predictability**: Ensures high consistency and controllability of AI outputs through structured instruction hierarchies and Few-Shot learning.
- **Chain-of-Thought Reasoning**: Utilizes CoT patterns to enable Agents to perform step-by-step reasoning before executing complex logic, significantly improving accuracy.
- **Agent-Native Optimization**: Optimized for the Agent's "working memory" (context window), maximizing information density while minimizing token usage.
- **Persuasion Psychology**: Applies principles like Authority and Commitment to ensure Agents strictly follow critical protocols (like TDD) even in complex scenarios.

## Key Features & Workflow

1. **Advanced Pattern Library**: Includes mature prompt patterns such as Few-Shot, CoT, Progressive Disclosure, and Instruction Hierarchy.
2. **Agent Interaction Best Practices**: Provides strategies for context window management, degrees of freedom settings, and error recovery.
3. **Persuasive Instruction Design**: Methods for writing "binding" instructions using psychological principles to reduce Agent "hesitation" or "shortcut-taking."
4. **Validation & Iteration**: Includes designs for self-verification steps, performance optimization (latency vs. cost), and RAG integration patterns.

## Skills Library Overview

- **Prompting Patterns**: A complete methodology ranging from basic instructions to complex template systems.
- **Persuasion Principles**: Analysis of 7 key persuasion principles for prompts to increase instruction compliance.
- **Performance Guide**: Optimization tips for token efficiency, latency reduction, and streaming modes.
