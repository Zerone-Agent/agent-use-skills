# Install Skill Market (Codex)

This guide describes how to configure **Skill Market** in Codex.

## Installation Steps

1. **Prerequisites**:
   Install the `uv` CLI tool.

2. **Verify Skill**:
   Run in the terminal:
   ```bash
   uv run awesome-skills/skills/skill-market/scripts/market.py list
   ```
