# Install Skill Market (Qoder)

This guide describes how to configure **Skill Market** in Qoder.

## Installation Steps

1. **Prerequisites**:
   Install the `uv` tool.

2. **Verify Test**:
   Run:
   ```bash
   uv run awesome-skills/skills/skill-market/scripts/market.py list
   ```

Reference: [Zerone Agent SDK](https://github.com/zerone-agent/agent-use-skills)
