# 安装 Skill Market (OpenClaw)

本指南介绍如何在 OpenClaw 中配置 **Skill Market**。

## 安装步骤

1. **环境依赖**：
   确保你的 Python 环境可以使用 `uv`。

2. **验证**：
   执行：
   ```bash
   uv run awesome-skills/skills/skill-market/scripts/market.py list
   ```
