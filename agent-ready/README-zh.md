## 什么是 Agent-Use？

**Agent-Use** 是 AI Agent 与软件生态系统之间的桥梁。我们正在构建基础架构，使传统软件能够实现“Agent-First”（Agent 优先）—— 从底层设计开始就支持被 AI 自主操作。

我们的愿景：在不久的将来，软件将不再仅仅是为人类交互而设计的。主要的接口将是语义化的、结构化的和可编程的。Agent-Use 正在让这一转变成为可能。

---

## 什么是 Agent-Ready？

**Agent-Ready** 是我们为希望在 Agent 优先时代蓬勃发展的传统软件厂商提供的认证计划。

一个通过 Agent-Ready 认证的软件产品具备：
- **原子化动作开放：** 核心功能通过标准化的 API（WebMCP、CLI 或 OpenAPI）可访问。
- **结构化反馈：** 执行结果以结构化日志形式输出，而不仅是视觉上的 UI 变化。
- **安全护栏：** 具备完善的权限边界和“人工在环”（human-in-the-loop）审批机制。
- **经过验证的集成：** 软件已经过 Agent-Use 的测试和认证，可与主流 AI Agent 平台无缝协作。

## 面向软件厂商

如果你代表一家传统软件公司，并希望让你的产品能够被不断增长的 AI Agent 生态系统所使用，Agent-Use 提供：

1. **咨询：** 关于如何为 Agent 消费而重构软件接口的专家指导。
2. **认证：** 官方的“Agent-Ready”标识，向开发者和 Agent 发出信号，表明你的软件是值得信赖且易于集成的。
3. **分发：** 在 Agent-Use Marketplace 上获得推荐，开发者在那里积极寻找高性能的工具。
